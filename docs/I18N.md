# I18N — FR/EN

## Exigence
- Switch langue visible dans la navbar (FR / EN)
- Persister choix (localStorage/cookie)
- Détection initiale possible via navigator.language (fr par défaut)

## Pages & UI à traduire
- Navigation: Accueil, Boutique, Packs, Personnaliser, Panier, FAQ, Contact
- CTA: Voir les packs, Composer mon pack, Ajouter au panier, Commander sur WhatsApp
- Labels formulaire, erreurs, région/devise, livraison, délais

## Stratégie recommandée
- UI: fichiers messages `messages/fr.json` et `messages/en.json`
- Produits: stocker `name.fr`, `name.en`, `desc.fr`, `desc.en` directement dans la data catalogue
  (plus simple, évite 300 clés i18n)
